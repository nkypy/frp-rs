# tfrp

### What is tfrp?
tfrp is a fast reverse proxy powered by rust lang.

### Usage
rename and edit config files
#### Server
```rust
cargo run --bin tfrps
```

#### Client
```rust
cargo run --bin tfrpc
```

### Notice
this project is created for learning rust and frp, please do not use in production.
